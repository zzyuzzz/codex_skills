# Novel Writing Skill — 网络小说拆解与创作指南

## Overview

This skill distills techniques from the acclaimed Chinese web novel *恐怖修仙世界* (2073 chapters, ~4.85M characters) into reusable writing patterns. The novel is a benchmark for **克苏鲁修仙** (Lovecraftian cultivation) — fusing cosmic horror with traditional xianxia progression.

Use this skill when creating ANY web novel, not just horror-cultivation. The structural patterns, tension management, and chapter hook techniques are genre-agnostic.

---

## 1. Opening Technique — 开篇钩子

### Pattern: "Disoriented Arrival + Immediate Threat"

Chapter 1 opens with the protagonist already **3 days post-transmigration**, disoriented, with memory loss. This avoids the stale "woke up in a new world" preamble. Key elements:

- **Physical vulnerability**: protagonist is injured (head wound), can barely walk
- **Atmospheric dread**: darkness triggers inexplicable fear (foreshadowing the supernatural)
- **Everyday detail as world-building**: clay-and-straw walls, oil lamps, coarse clothing — show poverty via sensory detail
- **Mini-crisis at chapter end**: door opens by itself, no one enters → cliffhanger

### Chapter Hook Rules (每章结尾必杀技)

Every chapter must end with one of these:
1. **Revelation hook**: New info that changes the stakes (e.g., "心口浮现的血色字数")
2. **Cliffhanger action**: Mid-action cut (e.g., door opens, hands around throat)
3. **Question hook**: A question left unanswered (e.g., "这是什么朝代？")
4. **False resolution**: Seeming safety, then twist (e.g., parents think it's fine, but the horror returns)
5. **Threat escalation**: The known danger gets worse

## 2. World-Building — 世界观递进披露

### Golden Rule: Never info-dump

The novel reveals its world through **experience, not explanation**.

| Layer | When Revealed | What's Introduced |
|---|---|---|
| Layer 1 (Ch1-5) | Protagonist's home | Dark fear, poverty, "阴鬼", oil lamps,卫鼓 |
| Layer 2 (Ch5-15) | 束发日 ceremony | 寿命数字,短命种 vs 长寿种, 寿鬼 |
| Layer 3 (Ch15-30) | 巡逻队 | 怪谲 types, 卫鼓 protection, 钓鱼 system |
| Layer 4 (Ch30+) | Village politics | Human threats, 符师/力士, cultivation paths |
| Layer 5+ | Each new region | Gradual world expansion |

### Information Leakage Techniques:
- **Through conflict**: Teach world rules during life-or-death moments
- **Through character**: Different characters explain different facets (father knows farming+ghosts, captain knows combat, etc.)
- **Through cost**: The protagonist pays a price to learn each new rule
- **Through mystery**: Leave gaps, let readers wonder

## 3. Tension & Horror Atmosphere — 恐怖氛围营造

### Core Horror Techniques

1. **Darkness as active threat**: Not just absence of light, but *something in the dark*
2. **Invisible menace**: Threats the protagonist cannot see (阴鬼 chokes him, but hands are invisible)
3. **Physical symptom → supernatural cause**: Fear response that seems psychological but is actually real
4. **Ordinary settings → corrupted**: A village, a house, a field — make the familiar terrifying
5. **Helplessness**: The protagonist survives by luck or others' help at first, not by power

### Tension Rhythm (张力节奏)
```
Build → Release → Build harder
```
Example arc in first 30 chapters:
- Ch1-3: Horror buildup (darkness,阴鬼 attack) → saved by parents
- Ch4-6: Brief calm → 束发日 → **寿命数字 revelation** (bigger horror)
- Ch7-10: New threat (短命种, must join巡逻队) → even bigger stakes
- Ch11-15: Training montage (relative calm) → 灰雾空间 reveal

Each calm period is shorter than the last; each crisis is worse.

## 4. Progression System — 成长体系设计

### Two-Track Progression

| Track | Resource | Progression |
|---|---|---|
| **武道 (Martial)** | Training + combat | 力气段→速度段→抗击段→技术段→意志段 |
| **修士 (Cultivation)** | 功法 + 丹药 | 通玄境→洗髓→换血→合道→超脱 |

### The "钓鱼" (Fishing) Goldfinger Design

The core progression mechanic has these properties:

1. **Cost**: Requires 灰虫 (earned by killing monsters) — links to combat
2. **Risk/Reward**: Different钓竿 give different categories (丹药, 功法, weapons)
3. **Scarcity**: Limited fishing attempts per visit — forces choices
4. **Upgrade Path**: The system itself can be improved (better rods, better bait)
5. **Mystery**: The source (船, 雾) is unexplained → drives curiosity
6. **Gatekeeping**: Certain items are locked until protagonist reaches certain thresholds

### Design Principles for Any Progression System:
- **Cost must tie to core gameplay loop**
- **RNG with player agency** (choose which rod, but not exactly what you get)
- **Always introduce the mechanic before it's needed** (fishing shown early, becomes critical later)
- **Progression must feel earned**, not gifted

## 5. Character Design — 人物塑造

### Protagonist: 周凡

**Core traits**: Cautious, analytical, former police officer (sharp observation), survivor's mindset

**Character flaws**: Overly suspicious, struggles to trust, carries past trauma

**Growth arc**: 
- Survival mode → building relationships → taking responsibility → protecting others → self-actualization

### Supporting Character Templates

| Type | Example | Function | Technique |
|---|---|---|---|
| **Mentor** | 雾 (Mist) | Guide with mystery | Never fully explain; keep ambiguous |
| **Partner** | 小柳/柳生花 | Equal-level companion | Grow together, mutual reliance |
| **Comic Relief** | 奥公公 | Tension breaker | Appear in high-stress moments |
| **Rival** | Various | Pacing pressure | Show what protagonist *could* become |
| **Love Interest** | 李九月/虫娘 | Emotional anchor | Give protagonist something to protect |
| **Family** | 周一木/桂凤 | Moral foundation | Humble origins, emotional stakes |

### Key Technique: "Companions with Secrets"
Every long-term companion has a hidden background that is gradually revealed:
- 小柳 → 谲人 (half-monster)
- 李九月 → 虫娘 (mysterious powerful background)
- 瘦猴 → secret past

This creates **re-reading value** and ongoing suspense.

## 6. Plot Architecture — 剧情架构

### Volume Structure (2073 chapters)

| Volume | Chapters | Theme |
|---|---|---|
| **新手村篇** | 1-149 | Survival horror, learning world rules |
| **天凉城篇** | 150-399 | Urban exploration, bureaucracy, guilds |
| **镜都篇** | 400-799 | Court intrigue, nation-level threats |
| **蛮星界篇** | 800-1199 | Multi-world expansion, cosmic scale |
| **星界海篇** | 1200-1599 | Travel arc, world-hopping |
| **混沌树篇** | 1600-1899 | Origin story, truth of universe |
| **终局篇** | 1900-2073 | Final battle, resolution |

### Arc Design Pattern:
```
Mini-Arc (20-50 chapters) = One major threat resolved
Mid-Arc (100-200 chapters) = New region + new system layer
Mega-Arc (300-500 chapters) = Full power tier + world-scale revelation
```

### Pacing Formula:
- **Action chapters** (30%): Combat, escape, crisis
- **Development chapters** (30%): Training, fishing, upgrades
- **Setup chapters** (25%): World-building, character moments, foreshadowing
- **Payoff chapters** (15%): Revelation, emotional climax, arc completion

## 7. Writing Technique — 文笔技法

### Sentence Rhythm
- **Short sentences for action**: Crisp, fast
- **Long sentences for atmosphere**: Layered descriptions of darkness, fear
- **Paragraph breaks**: Frequent breaks (~3-5 sentences) for mobile reading

### Sensory Detail
```
Bad: "The room was dark."
Good: "黄泥与干草混合搭建的房墙上只有一扇小窗，窗内投进一束光，屋顶的天窗也有白光泄进来，尘埃在光线内微微荡漾。"
```
Use at least **2 sensory modalities** per descriptive passage.

### Internal Monologue
- Protagonist constantly analyzes, deduces, plans
- Shows his intelligence without telling
- Creates reader identification (they think alongside him)

### Dialogue Style
- **Short exchanges** (rarely more than 5 lines per turn)
- **Subtext-heavy**: Characters rarely say exactly what they mean
- **Information asymmetry**: Each character knows different things

## 8. Horror-Specific Techniques — 恐怖专项技法

### The "Invisible Threat" Pattern
1. Protagonist senses something wrong
2. Physical evidence appears (marks, sounds, temperature changes)
3. Attack happens from unseen source
4. Near-death escape
5. Aftermath: explanation from others (partial)

### The "Normal Becoming Wrong" Pattern
1. Establish normal (a village, a house, a routine)
2. Small irregularities (door left open, a villager acting strange)
3. Escalation of weirdness
4. Full horror reveal
5. The normal is never restored

### Fear Sources Hierarchy:
| Level | Examples | Impact |
|---|---|---|
| Physical fear | Pain, injury, death | Immediate tension |
| Existential fear | 寿数, unknown fate | Long-term dread |
| Cosmic fear | The world itself is wrong | Permanent unease |
| Moral fear | Humans doing worse than monsters | Emotional impact |

## 9. Common Pitfalls to Avoid — 避坑指南

1. **Don't explain everything**: Keep some mysteries (the船 is never fully explained)
2. **Don't make protagonist too strong too fast**: 周凡 struggles for hundreds of chapters
3. **Don't forget the horror**: Even when power grows, keep the world dangerous
4. **Don't let romance dominate**: Subplot, not main plot
5. **Don't ignore side characters**: Give them arcs, don't discard after introduction
6. **Don't lose the stakes**: Each new power level should face commensurate threats

## 10. Chapter-Level Checklist — 章节检查清单

Before finalizing a chapter, verify:

- [ ] Ends with a hook (revelation/cliffhanger/question/threat escalation)?
- [ ] Every scene advances plot OR develops character?
- [ ] Sensory details in descriptive passages (≥2 senses)?
- [ ] Dialogue reveals character or advances plot (not filler)?
- [ ] Internal monologue shows protagonist's reasoning?
- [ ] At least one moment of tension/fear/unease?
- [ ] World-building revealed naturally (no info-dump)?
- [ ] Chapter title is intriguing but not spoilery?
- [ ] Word count ~2000-3000 characters per chapter (web novel standard)?
- [ ] If action scene: clear spatial layout, short sentences, high stakes?

## 11. Quick Start Template — 快速创作模板

### New Novel Outline Template:
1. **Core hook** (1 sentence): e.g., "主角发现自己心口有寿命倒计时"
2. **Opening scene**: protagonist in vulnerable state, immediate mini-crisis
3. **World's first rule**: revealed through first crisis survival
4. **Goldfinger hint**: shown but not explained in first 3 chapters
5. **First major choice**: forced by chapter 10, with irreversible consequences
6. **First power gain**: hard-won, not free, around chapter 15-20
7. **First full arc outline**: 20-50 chapters, one major threat, clear victory at cost

### Chapter Outline Template:
```
[Opening] Brief recap or immediate situation (1-2 paragraphs)
[Setup] New information or scene transition (2-3 paragraphs)
[Conflict] Tension rises, obstacle appears (5-10 paragraphs)
[Crisis] Worst moment, protagonist must make choice (3-5 paragraphs)
[Resolution/Payoff] Outcome + new question (1-3 paragraphs)
[Hook] Final line that makes reader click "next chapter"
```
