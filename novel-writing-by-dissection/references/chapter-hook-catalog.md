# Chapter Hook Catalog — 章节钩子目录

Hook patterns extracted from 恐怖修仙世界, organized by type.

## Type 1: Revelation Hook (揭示钩)

The chapter ends with new information that changes the reader's understanding.

### Examples:
- "心口浮现的寿数就像一个计时炸弹" — lifetime is counting down
- "你只有十九年的寿命" — concrete death sentence
- "这艘船上的钓竿..." — a new power system revealed
- "它不是死了，而是...蜕变了" — monster evolution twist
- "原来，这个世界是可以离开的" — new possibility opens

### Formula:
```
[Build-up to a discovery] → [Character makes/finds discovery] → 
[Full implication not yet shown, but hinted] → [Chapter ends]
```

## Type 2: Cliffhanger Action (动作钩)

Cut mid-action, resolution in next chapter.

### Examples:
- "木门被推开了，但没有脚步声传来" — door opens, something enters
- "双手掐住了周凡的脖子" — sudden attack
- "刀锋已至眼前" — imminent death
- "身后传来一声巨响" — off-screen event
- "他看到了一个人影，但那个人...没有脸" — horror reveal

### Formula:
```
[Action builds to peak] → [Critical moment] → [Chapter ends]
[Next chapter opens with protagonist reacting to peak moment]
```

## Type 3: Question Hook (疑问钩)

End with a question (explicit or implicit) that demands answer.

### Examples:
- "这个世界到底是什么样的？" — broad mystery
- "为什么偏偏是我？" — existential
- "那个人为什么要帮我？" — character motivation
- "这真的是巧合吗？" — suspicion
- "他还能活多久？" — fate question

### Formula:
```
[Character observes/experiences something inconsistent] → 
[Recognizes a gap in understanding] → [Chapter ends with implied question]
```

## Type 4: False Resolution (假安全钩)

Seems safe, then last line reveals danger.

### Examples:
- "应该没事了" (parents think danger passed) → BUT reader knows it hasn't
- "终于安全了" → BUT something follows
- "他们相信了我" → BUT someone is watching
- "这一切终于结束了" → BUT it was only the beginning

### Formula:
```
[Threat apparently resolved] → [Character relaxes] → 
[Final line hints threat continues or was a decoy]
```

## Type 5: Threat Escalation (威胁升级钩)

The known danger gets worse or a new layer appears.

### Examples:
- Not just阴鬼, but "寿鬼会在寿数尽时到来" — deadline added
- Not just monsters, but "人类也可以比怪物更可怕" — moral horror
- Not just this village, but "整个天凉里都陷入了危险" — scale increase
- Not just this world, but "这个世界之外还有更大的恐怖" — cosmic scale

### Formula:
```
[Current threat established] → [New information reveals it is worse than thought] →
[Chapter ends with character realizing new scope of danger]
```

## Type 6: Emotional Hook (情感钩)

End on an emotional beat that resonates.

### Examples:
- "爹，娘，对不起..." — guilt
- "如果能活着回去..." — hope
- "原来，我还是这么弱小" — frustration
- "这一次，我不会再逃了" — determination
- "再见了" — farewell/ loss

### Formula:
```
[Emotional buildup through the chapter] → [Character reaches emotional turning point] →
[Short, punchy final line expressing the emotion]
```

## Hook Placement Rules

1. **Always** end with a hook — never let a chapter fizzle out
2. **Vary hook types** — don't use the same type 3 chapters in a row
3. **Match hook to chapter content** — action chapter → action hook
4. **The best hooks** combine types (e.g., revelation + threat escalation)
5. **Multi-chapter hooks** — some hooks span 2-3 chapters, but still end each chapter with a mini-hook

## Anti-Patterns

- DON'T end with "欲知后事如何，且听下回分解" — too formulaic
- DON'T fake the hook (character "sees something shocking" but next chapter shows it was nothing)
- DON'T hook every chapter on the same threat
- DON'T resolve the hook too quickly (let it breathe 1-2 paragraphs into next chapter)
