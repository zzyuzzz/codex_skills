# Power System Reference — 修炼体系参考

## Stage 1: 凡人武道 (Mortal Martial Path)

### 力气段 (Strength Stage)
| Sub-stage | Description | Chapter Range |
|---|---|---|
| 力气初段 | Beginner strength training | ~Ch11-30 |
| 力气中段 | Intermediate | ~Ch30-70 |
| 力气中段极限 | Peak before breakthrough | ~Ch77-80 |
| 力气高段 | Advanced | ~Ch80+ |

### 速度段 (Speed Stage)
| Sub-stage | Description |
|---|---|
| 速度初段 | Begin speed training |
| 速度段极限 | Speed peak |

### 抗击段 (Endurance Stage)
- 抗击中段, 抗击高段

### 技术段 (Technique Stage)
- Weapon mastery, combat technique

### 意志段 (Will Stage)
- Mental fortitude, willpower training

## Stage 2: 修士修真 (Cultivation Path)

### Core Cultivation Stages
1. **通玄境 (Mystery Penetration)**:洗髓→换血→换骨→...
2. **合道境 (Dao Integration)**: Multiple sub-stages
3. **超脱境 (Transcendence)**: Beyond normal cultivation
4. **混沌主宰 (Chaos Master)**: Final evolved state
